            Content schedule tips

            • Bible Quiz: set the show date in the Bible Quiz form so a new question unlocks from midnight Belgium time that day (weekdays in the app).
            • Events: add dates here; they appear on Home and the Events tab.
            • Notices: publish to MantleDB; they merge into Home announcements with the static list.
            • Daily bread: add dated entries; the Daily bread tab shows the entry for today (Brussels date). Create the MantleDB bucket njc-belgium-admin-daily-bread if needed.
